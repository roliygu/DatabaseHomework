这个项目是我数据库课程的大作业，题目是成绩录入系统（成绩管理数据库）
初步的功能：
	1.录入数据，从文本中读取数据，转存到数据库；
	2.使用Python将一些常用的SQL语句封装成函数，起到简化查询的作用；
	3.用户管理子系统，类似于权限管理，这里打算用Python做一个中间层，每次使用此程序时先用户登录，更新用户权限。

流程和计划：
	1.熟悉Python和Mysql的链接和嵌入式查询；
	2.设计数据库结构；
	3.用户子系统开发。

开发环境与工具：
	Python 2.7 ， MySQL 和 Git版本管理器。

第一阶段：
	起于24日8点，熟悉Python和MySQL的交互
