这个项目是我数据库课程的大作业，题目是成绩录入系统（成绩管理数据库）
初步的功能：
	1.录入数据，从文本中读取数据，转存到数据库；
	2.使用Python将一些常用的SQL语句封装成函数，起到简化查询的作用；
	3.用户管理子系统，类似于权限管理，这里打算用Python做一个中间层，每次使用此程序时先用户登录，更新用户权限。

流程和计划：
	1.熟悉Python和Mysql的链接和嵌入式查询；
	2.设计数据库结构；
	3.用户子系统开发。

开发环境与工具：
	Python 2.7 ， MySQL ，亿图图示专家和Git版本管理器。

数据库结构：
	分为三张table：
	1.学生table，用于存储学生信息，包括
		学号		varchar(7)    
		姓名		varchar(20)
		专业		varchar(20)
		primary key (学号)；
	2.take-table，这个是学生table和课程table的链接表。包括，
		成绩		numeric(3,0)
		学号		varchar(7)
		课程号  	varchar(8)
		年份		numeric(4,0)
		primary key （学号，课程号，年份）；	
	3.课程table，用于存储课程信息，包括，
		课程号		varchar(8)	
		课程名		varchar(20)
		课程专业 	varchar(20)
		primary key （课程号）；

用户子系统：
	1.密码函数
